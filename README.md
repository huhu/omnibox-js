# omnibox-js

Turn your `div` label into chrome `omnibox`.


## Example

```
$ npx serve
   ┌───────────────────────────────────────────┐
   │                                           │
   │   Serving!                                │
   │                                           │
   │   - Local:    http://localhost:3000       │
   │   - Network:  http://192.168.1.122:3000   │
   │                                           │
   │   Copied local address to clipboard!      │
   │                                           │
   └───────────────────────────────────────────┘

$ open http://localhost:3000/examples/
```
